# Legacy (архів)

Це **оригінальна, домодернізаційна** версія «Морського бою» — монолітний варіант:

- `index.html` — React + Babel через CDN (транспіляція JSX у браузері);
- `Progect.js` — увесь застосунок одним компонентом;
- `server.js` — Express + Socket.io (релей) + `sqlite3` (CommonJS);
- `style.css` — старі стилі.

**Не використовується і не запускається.** Збережено лише для історії. Актуальна версія —
у `client/` + `server/` + `shared/` (див. README в корені проєкту).
